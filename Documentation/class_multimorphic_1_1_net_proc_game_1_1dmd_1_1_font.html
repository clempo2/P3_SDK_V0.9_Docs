void load (string filename)
 Loads a font from a dmd file. Fonts are stored in .dmd files with frame 0 containing the bitmap data and frame 1 containing the character widths. 96 characters (32..127, ASCII printables) are stored in a 10x10 grid starting with space (' ') in the upper left at 0,0 The character widths are stored in the second frame within the raw bitmap data in bytes 0-95.

Detailed Description

Variable width bitmap font.

Fonts can be loaded manually, using the load() method or with the font_named() utility function which supports searching a font path.
