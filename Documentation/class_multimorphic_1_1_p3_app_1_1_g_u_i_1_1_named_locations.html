Detailed Description

A container for named locations in the scene. Named locations are in-scene locations which are relevant to specific elements of the game. For example, they could indicate places from which score popups could begin, or where they might drift towards. Named locations are defined by adding empy GameObjects objects as children of this script's GameObject. The names of these child objects will be used as the named locations. Subsequently, these positions can be retrieved by calling PositionByName.

Internally, this class finds and stores the named locations at Awake time, to eliminate object finding during the scene. Therefore, it's not suitable for moving named locations.
