Detailed Description

Allows the GUI to subscribe to events from the modes and to post events to the modes. Provides basic ball tracking data. Interprets key mappings. Starts the thread for the modes. Starts the switch recorder.

out Multimorphic.P3.Vectors.Vector2 gridPosition,
out Multimorphic.P3.Vectors.Vector2 velocity )

Gets the ball position and velocity from the ball tracker. Coordinates and velocity are within the infrared gridspace, which roughly correspond to inches but have no real relation to the scene. Such a relationship is achieved via BallAvatar (which makes use of this function). For almost all cases, the P3Aware class contains enough methods to detect interaction with the ball.

Returns
true, if ball position and velocity was retrieved, false otherwise.
Parameters

gridPositionGrid position.
