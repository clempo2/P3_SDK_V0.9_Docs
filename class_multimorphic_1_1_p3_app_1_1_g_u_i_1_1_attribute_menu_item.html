Find the point on the given plane which, from the camera's point of view, is "behind" or "underneath" a game object. In other words, if a line were projected from the camera through the game object's transform's origin, where on the given plane would it hit?

Returns
true if the projected point exists on the plane, false otherwise (in which case projectectPoint's value is meaningless).
Parameters
planeThe plane whose intersection is being detected.
pointThe object whose position will be used as the point through which a line will be projected from the camera.
projectedPointThe point where the line intersects the plane.
