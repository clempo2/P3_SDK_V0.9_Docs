Detailed Description

Public interface to the P3's trough ball launching logic. To use: First: call AssignP3 with with P3Controller object Next: call AssignTroughLauncher for each trough VUKNumber you might request a launch from. Give each one a unique KeyName. If you want to assign backup trough VUKNumbers in case a launch fails, call AssignTroughLauncher again with additional VUKNumbers for the same KeyName as the primary VUK.

You can assign the same VUKNumber to a primary KeyName and one or more backup KeyNames. Just be sure to assign the primary first.

When you want to request a ball launch from one of the assigned VUKNumbers, call the appropriate launch method with either the VUKNumber of the KeyName.
