void Multimorphic.P3App.Modes.Selector.SelectorManagerMode.RegisterSelector ( SelectorMode mode,
string dialogName,

string assetPackVersionAsString,

bool activateWhenDialogActivates = true )

Registers a selector for later use, identifiable by its selectorId. Also specifies the dialogId of the GUI object which contains the selector game object. Selectors should be a child of a parent gameObject which will likely contain a title, framing, etc. and perhaps other selectors. This parent is known as the dialog.

Parameters
modeThe mode which is being registered for later use. It's selectorId property must be set and must be unique.
dialogNameThe identifying name of the dialog, as it will be called in OpenDialog and CloseDialog
assetPackNameThe name of the asset pack that contains the dialog prefab.
assetPackVersionAsStringThe version of the asset pack that contains the dialog prefab. (Version.ToString())
assetBundleNameThe name of the asset bundle that contains the dialog prefab.
dialogPrefabNameThe path of the prefab which contains this selector.
activateWhenDialogActivatesSet to true if this selector should automatically activate when the dialog activates. Defaults to true so that dialogs which consist of a single selector can ignore this parameter. Set to false if the selector is not meant to be active when the dialog activates, which is likely to be the case for the second selector (and subsequent selectors) within a single dialog.
