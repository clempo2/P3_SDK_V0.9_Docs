Multimorphic.P3SA.GUI.P3SAAudio Class Reference

The implementation of audio for this project. Implements the Multimorphic.P3App.GUI.IAudio interface. Allows individual clips to be played directly or in 3D locations. Allows playlists (several sequential clips) to be specified and triggered. Playlists are added by adding a Playlist component to this object's gameObject (in the inspector) and adding clips to that Playlist. Allows volume control of the individial sound clips (as a group) and the playlists (as a group). More...

Detailed Description

The implementation of audio for this project. Implements the Multimorphic.P3App.GUI.IAudio interface. Allows individual clips to be played directly or in 3D locations. Allows playlists (several sequential clips) to be specified and triggered. Playlists are added by adding a Playlist component to this object's gameObject (in the inspector) and adding clips to that Playlist. Allows volume control of the individial sound clips (as a group) and the playlists (as a group).
