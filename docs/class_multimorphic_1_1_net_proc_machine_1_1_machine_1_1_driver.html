Detailed Description

Represents a driver in a pinball machine, such as a lamp, coil/solenoid, or flasher.

◆ Enable()

void Multimorphic.NetProcMachine.Machine.Driver.Enable ( )

Enables this driver indefinitely.

WARNING!!! Never use this method with high voltage drivers such as coils and flashers! Instead, use time-limited methods such as 'pulse' and 'schedule'.
