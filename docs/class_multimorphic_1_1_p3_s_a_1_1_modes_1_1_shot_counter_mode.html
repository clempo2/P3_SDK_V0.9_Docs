◆ AbortSoundsOnMixerGroup()

void Multimorphic.P3SA.Modes.P3SAGameMode.AbortSoundsOnMixerGroup ( string mixerGroupName,
bool interruptiveAbort = false )
protectedinherited

Abort all sounds on mixer group, with the option to hard or soft abort any sound that is currently playing.

Parameters
mixerGroupNameThe name of the mixer group to abort sounds on.
interruptiveAbortIf true, sound clips will be hard interrupted. If the sound clip is already playing, it will cut off in process. If false, any clip that is already playing will not be interrupted and will play to the clip's conclusion. Either way, any subsequent followups will not be played.
